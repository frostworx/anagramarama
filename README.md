# anagramarama

_( [original untouched readme](readme) )_

this is a quick emscripten port of the excellent SDL puzzle game anagramarama
the port ist based on this active fork

https://github.com/dulsi/anagramarama 

I made use of "Gemini (Google AI)" for the modifications.
This is more or less a born dead project, because I won't actively work on it.

I did not modify the cmake part. If you want to (re)compile the project you might want to use the hardcoded 'makey.sh'

You can play directly the bundled emscripten port here:

https://frostworx.github.io/anagramarama/

Have fun!
